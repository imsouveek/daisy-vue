# DaisyLoading

<Meta of={PrimaryStory} />

A loading component that uses animation to indicate that something is loading. This component
uses the [Daisy UI Loading component](https://daisyui.com/components/loading/) and exposes most of the
available options.

## Usage

The style of the laoder can be modified through various properties

### Colors

Use theme colors `primary`, `secondary`, etc. or brand colors such as `success`, `error` to control the color of the loader

<div className="p-8 border-1 rounded border-gray-200 shadow-sm shadow-gray-200 dark:shadow-none dark:border-gray-800 dark:bg-slate-900">
    <Story of={VariationsStories.Colors} />
</div>

### Sizes

Size of the loader can be controlled through this prop
