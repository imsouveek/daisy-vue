## Usage

This component can only be used with a list of `MenuItem` objects and a slot with no variations. Each menu item has the following properties:-

| Property   | Type         | Description                                                |
| ---------- | ------------ | ---------------------------------------------------------- |
| `label`    | `string`     | The item label to show in the menu                         |
| `shortcut` | `string`     | Keyboard shortcut for menu. This is displayed to the right |
| `action`   | `() => void` | The callback function on item click                        |
| `children` | `MenuItem[]` | Array of child menu items                                  |
