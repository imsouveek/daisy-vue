# DaisyAccordion

<Meta of={PrimaryStory} />

An accordion component. This component uses the [Daisy UI Accordion component](https://daisyui.com/components/accordion/)
and the [Daisy UI Collapse component](https://daisyui.com/components/collapse/) and exposes most of the available options.

## Usage

The behavior and icon of the accordion can be modified through various properties

### Icons

The icon of the accordion panels can be changed using the icon property

The accordion behavior can be modified to enable multiple panels to be expanded or all panels to be collapsed
