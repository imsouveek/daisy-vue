# DaisyButton

<Meta of={PrimaryStory} />

A flexible button component for to allow users to take action or make choices. This component
uses the [Daisy UI Button component](https://daisyui.com/components/button/) and exposes most of the
available options.

<Canvas of={PrimaryStory.Playground} />

<Controls />

## Usage

The style of the button can be modified through various properties

### Colors

Use theme colors `primary`, `secondary`, etc. or brand colors such as `success`, `error` to control the color of the button

<div className="p-8 border-1 rounded border-gray-200 shadow-sm dark:bg-slate-800">
    <Story of={VariationsStories.Colors} />
</div>

### Sizes

Size of the button can be controlled through this prop

<div className="p-8 border-1 rounded border-gray-200 shadow-sm dark:bg-slate-800">
    <Story of={VariationsStories.Sizes} />
</div>

### Other Variations

Buttons support a number of variations, such as disabled, square or circle shape, outline or dashed buttons, etc
