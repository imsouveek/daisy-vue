# DaisyAvatar

<Meta of={PrimaryStory} />

An avatar component. This component
uses the [Daisy UI Avatar component](https://daisyui.com/components/avatar/)
and exposes most of the available options.

## Usage

The style of the avatar can be modified through various properties

### Sizes

Size of the avatar can be controlled through this prop

<div className="p-8 border-1 rounded border-gray-200 shadow-sm shadow-gray-200 dark:shadow-none dark:border-gray-800 dark:bg-slate-900">
    <Story of={VariationsStories.Sizes} />
</div>

### Presence

Presence can be indicated using the presence prop. Note that this prop does not work inside an avatar group component

<div className="p-8 border-1 rounded border-gray-200 shadow-sm shadow-gray-200 dark:shadow-none dark:border-gray-800 dark:bg-slate-900">
    <Story of={VariationsStories.Presences} />
</div>

### Placeholder

Content for the avatar can be specified using either the placeholder prop to pass a string or content in the default slot

<div className="p-8 border-1 rounded border-gray-200 shadow-sm shadow-gray-200 dark:shadow-none dark:border-gray-800 dark:bg-slate-900">
    <Story of={VariationsStories.Placeholder} />
</div>

### Masks

The mask property can be used to set the shape of the avatar

<div className="p-8 border-1 rounded border-gray-200 shadow-sm shadow-gray-200 dark:shadow-none dark:border-gray-800 dark:bg-slate-900">
    <Story of={VariationsStories.Masks} />
</div>
